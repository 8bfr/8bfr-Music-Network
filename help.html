🔧 Common Issues

I can't log in

Make sure you're using the email you signed up with. Try the password reset if you forgot your password. Clear your browser cache if the page freezes.

My profile won't save

Make sure all required fields are filled. Avatar and banner images must be under 5MB. Try refreshing the page and saving again.

Chat messages aren't loading

This is usually a session timeout. Refresh the page — the token will auto-refresh. If it persists, log out and back in.

I can't upload music

Artists and beatmakers need to be verified first. Check your verification status on your profile. Audio files must be MP3 format.

Pages are freezing

If any page freezes on load, try clearing your browser cache. This was caused by an old script that has been removed from most pages. If it still happens, report which page to the owner.

I was muted in chat

Chat mutes are issued by staff for rule violations. Mutes can be temporary or permanent. Contact the owner to appeal.
