🛠️ Quick Troubleshooting

Can't log in? — Try resetting your password. Make sure you're using the email you signed up with.

Upload not working? — Max file size is 20MB for audio. Use MP3 format. Make sure you have the artist or beatmaker role.

Chat not loading? — Refresh the page. If still broken, clear your browser cache and try again.

Beat Store issues? — Make sure your PayPal email is set in Settings. Buyers pay you directly.

Missing features? — Some features require specific roles or the Verified badge. Check your profile badges.
